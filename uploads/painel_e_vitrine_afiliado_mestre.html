Criar campanha de rastreamento

Gere links com Sub IDs para Facebook, Instagram, WhatsApp etc.

Na puxada dos produtos, cada item já recebe Sub IDs no banco (afiliada_mestre | organico | vitrine | categoria | pID). Use o gerador abaixo quando for divulgar — o clique troca canal/campanha e identifica a origem.

Sem produtos = link geral da vitrine. Com produtos = um link por item (abre o produto certo).

Links para divulgar

1 afiliada_mestre → marca do site
2 canal → facebook / instagram / …
3 campanha → nome da campanha
4 categoria → no clique
5 produto → p + ID no clique

Campanhas salvas

Guarde links de Facebook, Instagram etc. e reabra quando for anunciar.

Como anunciar no Facebook / Meta
  1. Crie a campanha aqui, escolha canal Facebook e os produtos.
  2. Clique em Salvar campanha e copie o link.
  3. No Gerenciador de Anúncios Meta → criar anúncio → destino = Tráfego / site.
  4. Cole o link gerado no campo URL do site (não use o link cru da Shopee).
  5. Publique. Quem clicar e comprar leva seus Sub IDs.

O Sub ID aparece no relatório oficial da Shopee (campo utmContent / Sub IDs), desde que a pessoa use o botão Comprar da vitrine.
