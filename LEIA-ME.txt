Afiliado Mestre — setup rápido
==============================

1) No Supabase → SQL Editor → cole e rode o arquivo:
   sql/schema.sql

2) Neste pasta, no terminal:
   npm install
   npm start

3) Abra no navegador:
   http://localhost:3789

4) No painel (API & Console Admin) use:
   - "Buscar na Shopee" / sync para puxar productOfferV2
   - Ofertas vão para a tabela public.ofertas no Supabase
   - Vitrine lê o cache do banco (ou busca ao vivo)

IMPORTANTE
- SHOPEE_APP_ID / SHOPEE_SECRET ficam só no .env (nunca no HTML)
- SUPABASE_SERVICE_KEY também só no servidor
- Você colou as keys no chat: se esta conversa for compartilhada, rotate as keys no Supabase
- Botão "Comprar" abre offerLink da Shopee (não entra direto no carrinho — limitação da API)
